# AssetFlow ERP

Enterprise Asset & Resource Management System

## Problem Statement
AssetFlow is an ERP application for managing organizational assets, resource bookings, maintenance workflows, audits, and analytics.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express (planned)
- Database: MongoDB (planned)
